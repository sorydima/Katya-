# Katya® - Ultimate AI Multifunctional Social Blockchain Platform

Добро пожаловать в Katya®! Это основной README. Подробная документация доступна в [docs/README.md](docs/README.md).

## Быстрый старт

- **Клонировать репозиторий**: `git clone https://github.com/your-username/katya.git`
- **Установить зависимости**: `flutter pub get`
- **Запустить**: `flutter run`

## Структура проекта

- `docs/` - Документация и руководства
- `reports/` - Отчёты и changelog
- `research/` - Исследования и заметки
- Платформы: `android/`, `ios/`, `linux/`, `macos/`, `windows/`, `web/`, `winuwp/`, `aurora/`
