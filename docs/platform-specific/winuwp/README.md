# WinUWP Platform Documentation

This directory contains platform-specific documentation for Windows UWP builds and deployments.

## Build Instructions

- Run `flutter build winuwp --release` for release builds.
- Use `flutter run -d winuwp` for development.

## Requirements

- Windows 10 or higher with UWP support.
- Visual Studio 2019 or higher.
