# macOS Platform Documentation

This directory contains platform-specific documentation for macOS builds and deployments.

## Build Instructions

- Run `flutter build macos --release` for release builds.
- Use `flutter run -d macos` for development.

## Requirements

- macOS 10.14 or higher.
- Xcode 13.0 or higher.
